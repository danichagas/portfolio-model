# 🌐 Portfólio Model

Este é um projeto de um modelo de portfólio, desenvolvido para apresentar o perfil de alguma pessoa ou organização, as habilidades e os projetos construidos. 

O objetivo principal foi criar uma página estática, limpa e responsiva utilizando apenas tecnologias web fundamentais.

---

## 🚀 Funcionalidades e Estrutura

O site é composto por uma única página (Single Page) ou navegação estruturada dividida em três secções principais:

* **🏠 Home:** A página de boas-vindas, com uma breve introdução, o meu nome e uma apresentação inicial chamativa.
* **👤 Sobre:** Um espaço dedicado a contar um pouco mais sobre a jornada da pessoa, interesses e as tecnologias que utiliza.
* **💻 Projetos:** Uma galeria ou lista para expor os principais projetos que desenvolveu, com links ou descrições.

---

## 🛠️ Tecnologias Utilizadas

Para a construção deste projeto, utilizei apenas as tecnologias base da web:

* **HTML5:** Para a estruturação semântica de todo o conteúdo da página.
* **CSS3:** Para a estilização, definição de cores, tipografia, alinhamentos e responsividade (adaptação para ecrãs de telemóveis e computadores).
